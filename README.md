# Advanced-Java-Backend-System-Development-Projects

Project Description

A collection of advanced Java projects developed as part of coursework, covering key backend concepts including concurrency, network programming, microservices communication, and web application development.

The projects include:

Concurrent Programming Simulation – Implementation of a multithreaded system simulating parallel student defense sessions with synchronization constraints, thread coordination, and resource management.

Socket-Based Public Chat Application – Client-server real-time chat system using Java Sockets with username validation, message history storage (limited to 100 messages), server-side message filtering, and broadcast communication.

Microservice-Based Quotes Management System – Distributed system consisting of a main service and an auxiliary service communicating via HTTP over sockets. The system handles quote storage and dynamic retrieval of a “quote of the day” using inter-service communication.

Servlet-Based Food Selection Web Application – Web application built using Java Servlet API that manages session-based meal selection, file-based data loading, access control, and administrative order management.

These projects demonstrate strong understanding of backend architecture, networking, concurrency control, session management, and distributed system communication.
